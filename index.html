I have been working on my own game, themed from the Touhou Project. Here is a short demo. Changelog: November 9, 2017: v0.01 released, first demo release. Only two non-spells and two spellcards available.

Download (Windows Only)

Instructions:
1. Extract the .zip file. Open the executable titled "th_dnh.exe"
2. Use the arrow keys to highlight "Stage", then start the "Hong Meiling Fight". Use the Z key to confirm a selection.
Controls:
Arrow keys: move, select difficulty
Z: Shoot bullets
X: Use bomb
Shift: Go into focused mode (increased damage, slower movement, hitbox shown)
The game is balanced around using the included ReimuA, ReimuB, and ReimuC characters.
